## InFlow App

Inflow is a multi-tenant SaaS platform that enables organizations to create dynamic forms and automate approval workflows.

### Tech Stack

- Frontend: HTML, Tailwind, TypeScript, React, Zustand
- Backend: Java 17, Spring, Spring Boot, Spring Data JPA, Spring Security, JWT
- Database: PostgreSQL
- DevOps: Git, Docker, K8s, Github Actions for CI/CD, AWS
